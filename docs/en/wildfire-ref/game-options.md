Start game

- check your laptop and email
- login I.M.P. website with activation code: `cia003`
- create your own mercenary base on [I.M.P. Question](./attrs-skills-and-traits.md#imp-question-guide)

If you create own mercenary with Night Ops and Stealthy skills, the gears should include:

- Kevlar Helmet
- Guardian Vest
- Beretta 92F
- 9mm Pistol Magazine, AP x4
- Silencer for 9mm
- Camouflage Kit
- Canteen
- 1st Aid Kit
- Break Lights x4

Suggested bonus points allocation:

- Health 90
- Dexterity 90
- Agility 90
- Strong 90
- Wisdom 90
- Leadership 35
- Marksmanship 75
- Explosive 0
- Mechanical 0
- Medical 0
