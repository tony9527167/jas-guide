# Tony's Jagged Alliance 2: Wildfire Guide Reference

Jagged Alliance 2: Wildfire was published in 2004 an official expansion pack of Jagged Alliance 2.
Comparing to the original Jagged Alliance 2, Wildfire was instead directed designing revamped environments, new items and stronger enemies.

## Where to obtain a copy

- [GOG copy](https://www.gog.com/en/game/jagged_alliance_2_wildfire) version 6.04 (build 12.01.06)
- [Steam copy](https://store.steampowered.com/app/215930/Jagged_Alliance_2__Wildfire/) version 6.04 (build 12.01.06)

**NOTICE** : Both GOG and Steam versions are buggy and have compatibility issues.
For more details, [see here](./bugs.md) .

### GOG Copy System Requirements

Minimum system requirements For Windows:

- System: Windows 7 or 10.
- Processor: 1.4 GHz or faster
- Memory: 512 MB RAM
- Graphics: 3D graphics card compatible with DirectX 9.0c

Please be advised that Windows 10 operating system will receive frequent hardware driver and software updates following its release; this may affect game compatibility

**Best system requirements For Windows** :

- OS: Windows 7
- Monitor: 4K resolution
- Hard Drive: SSD

For Linux:

- System: Ubuntu 14.04, 16.04 or later
- Processor: 1.6 Ghz Intel Core 2 Duo
- Memory: 1 GB
- Graphics: 512 MB of video RAM
