## [ Gap-File-Generator ]

2.09 MB | Downloads: 2188

This program generates .gap files. These files are used by JA2 to synchronize lip movement with speech. Without these files, NPCs still function normally, but their mouths move constantly.

For version: all

By [TrailBlazer](mailto:trailblazer_lotb@yahoo.gr)
