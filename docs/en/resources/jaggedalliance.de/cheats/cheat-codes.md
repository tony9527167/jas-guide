# cheat-codes

**Vorwort** :

Bevor ihr zum letzten Mittel greift - den cheat-codes - solltet ihr versuchen das Spiel so durchzuspielen. Ihr verderbt euch sonst nur den ganzen Spass!

Versucht lieber, wenn ihr an einer Stelle selbst im leichtesten Schwierigkeitsgrad mal nicht gleich weiterkommt, euch durchzubeißen. Mit ein wenig Geduld sollte alles problemlos zu schaffen sein. Sollten denoch alle Stricke reissen findet ihr immer in einem der Ja2 Foren Hilfe von anderen Fans. Schaut z.B. einfach mal in das [Ja 2 Basis Forum], meistens werdet ihr dort schon eine Antwort auf eure Frage finden (mit der "suchen" Funktion) oder inerhalb kürzester Zeit eine Antwort auf eure Frage bekommen.

Beachtet auch, dass die Benutzung der Cheats in das Logbuch eingetragen wird und daher jederzeit nachgewiesen werden kann...

Ach ja, die Benutzung der cheat-codes geschieht natürlich auf eigene Gefahr...

**Anleitung** :

Zuerst müsst ihr den Cheat-Modus aktivieren. Wechselt dazu in den Taktikbildschirm und gebt STRG + IGUANA ein (STRG-Taste dabei gedrückt halten).

Danach könnt ihr folgende Cheats ausführen:

**Cheat-Codes im Taktikbildschirm** :

- `ALT + E` Alle sich im Sektor befindlichen Einheiten (Gegner, NPCs, Zivilisten etc.) und Gegenstände werden sichtbar.
- `ALT + T` Teleportiert den ausgewählten bzw. aktiven Söldner an den ort des Cursors.
- `ALT + O` Alle gegnerischen Einheiten im Sektor werden eliminiert, mit Ausnahme von neutralen Zivilisten und Panzern. Bloodcats und Bugs werden erst nach mehrmaliger Eingabe des Cheats getötet.
- `ALT + D` Die Aktionspunkte des aktiven Söldners werden wieder auf die Maximalzahl aufgefrischt.
- `ALT + R` Die verwendete Waffe des aktiven Söldners wird wieder voll aufgeladen.
- `ALT + W` Sie können durch die Liste aller verfügbaren Gegenstände "scrollen" und dem aktiven Söldner einen ausgesuchten Gegenstand in die Hand geben. Wichtig dabei: Sie müssen sich in der Detailansicht eines Gegenstandes befinden (rechter Mausklick auf einen Gegenstand im Inventar).

- `ALT + B` Ein Gegner erscheint an der Stelle des Cursors.
- `ALT + C` Ein neutraler Zivilist erscheint an der Stelle des Cursors.
- `ALT + K` Eine Senfgas Granate explodiert an der Stelle des Cursors.
- `ALT + I` Ein zufällig ausgewählter Gegenstand erscheint an der Stelle des Cursors.
- `ALT + Q` Alle Gebäude werden aufgedeckt, so dass man hineinschauen kann.
- `ALT + G` Ein Söldner (A.I.M., M.E.R.C. oder RPC) erscheint an der Stelle des Cursors. Solange möglich, bis die Söldnerliste voll ist
- `ALT + Y` Der Roboter erscheint an der Stelle des Cursors.
- `ALT + 2` Der aktive Söldner verwandelt sich in einen mittelgroßen Bug.
- `ALT + 4` Der aktive Söldner verwandelt sich in einen Rollstuhlfahrer.
- `ALT + 5` Der aktive Söldner verwandelt sich in einen großen Bug.

- `STRG + O` Ein feindlicher Bug erscheint an der Stelle des Cursors.
- `STRG + K` Eine Granatenexplosion erscheint an der Stelle des Cursors.
- `STRG + T` Die Söldner im Sektor werden von der Königin gefangengenommen.
- `STRG + H` Befindet sich der Cursor über einem Söldner/Gegner verliert dieser Gesundheitspunkte.
- `STRG + U` Befindet sich der Cursor über einem Söldner/Gegner wird dieser Geheilt. Er erhält maximale Gesundheit und kurzeitig volle Energie.
- `STRG + M` Zeigt den vom Spiel verwendeten Speicher an.

**Cheat-Codes im Kartenbildschirm** :

- `STRG + T` Teleportiert die gewählte bzw aktive söldnergruppe in einen beliebigen Sektor. Ihr müsst dazu im Reise-Planungs-Modus sein, so dass die Marschroute sichtbar ist.
- `ALT + l.M.` Mit ALT + linker Mausklick auf das "Gehe zu Sektor" Icon im Feindkontaktbildschirm klicken. Alle Gegner in dem Sektor, in dem der Feindkontakt statfindet, werden eliminiert.

**Cheat-Codes in der Laptop-Ansicht** :

`+` Zusätzliche 100.000$ aufs Konto.
`-` 100.000$ weniger auf dem Konto.
