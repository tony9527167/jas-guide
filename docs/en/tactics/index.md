# JA2 Tactical Doctrine - Overview

"Tactical Doctrine" isn't meant to imply that there is only one correct way to play. After thousands of hours with JA2 and especially Wildfire, I actually ended up discovering that there are many very different viable styles, for example:

- rushing the game as fast as possible by going straight for the Queen
- playing a long guerrilla-style campaign
- using "pocket tactics" with small, highly mobile squads
- or even never finishing the game at all and just controlling all of Arulco
