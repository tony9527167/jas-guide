# Contributing

This project accepts:

- Bug reports
- Typos
- Factual corrections
- Additional test results
- Source documentation

It does NOT accept:

- Rewrites
- Tone changes
- Opinion edits
- Play-style preferences

All contributions are reviewed for:

- Accuracy
- Version correctness
- Mechanical validity

By submitting a contribution, you agree that it may be included in the canonical archive under CC BY-NC-ND 4.0.
