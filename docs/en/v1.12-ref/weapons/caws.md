# JA2 v1.12 Reference - CAWS

CAWS

This German/US Experimental Close Assault Weapon System makes the concept of human skeet shooting a gory reality.

Pros: high damage  
Cons: heavy, short range, unreliable, hard to repair

| Attr     | Value      |
| -------- | ---------- |
| Mag\*Cal | 10 \* CAWS |
| Weight   | 5.1        |
| DAM      | 40         |
| RNG      | 13         |
| AP       | 9/13       |
| Burst    | 3          |

Location:

- I6 Estoni Maddog

Accessories

- [Rod & Spring](./rod-and-spring.md)
- Duckbill
- Bipod
- Laser Scope

The _CAWS_ is short from Close Assault Weapon System.

NOTICE: This is one of the rarest weapons in the game, and also one of Maddog's default gear.
