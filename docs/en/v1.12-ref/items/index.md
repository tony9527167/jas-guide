## Item Merge

Firearm Enhancement

- Shotgun + Duckbill
- assault rifle + Sniper Scope + silencer + Bipod
- assault rifle + Talon + 40mm grenade
- handgun/rifle + Laser Scope

Explosives

- C4 = C1 + RDX
- HMX = TNT + RDX
- remote control bomb = Remote Detonator + C1 / C4 / HMX / TNT
- time bomb = Detonator + C1 / C4 / HMX / TNT

Armor Enhancement

- enhanced armor xx = Ceramic Plates + Flak Jacket / Leather & Kevlar Jacket / Spectra Vest / Kevlar Vest

- Compound 18 xx = Compound 18 + Flak Jacket / Leather & Kevlar Jacket / Guardian Vest
- Compound 18 Kevlar xx = Compound 18 + Kevlar Helmet / Kevlar Vest / Kevlar Leggings
- Compound 18 Spectra xx = Compound 18 + Spectra Helmet / Spectra Vest / Spectra Leggings

Armor Enhancement (sci-fi mode)

- Coated xx = Royal Jelly + Flak Jacket / Leather & Kevlar Jacket / Guardian Vest
- Coated Kevlar xx = Royal Jelly + Kevlar Helmet / Kevlar Vest / Kevlar Leggings
- Coated Spectra xx = Royal Jelly + Spectra Helmet / Spectra Vest / Spectra Leggings

X-Ray Detector

- [X-Ray Detector](./x-ray-detector.md) = X-Ray Device + Display Unit + Batteries
- X-Ray Device = FumblePak + X-Ray Tube + Pack of Gum
- Display Unit = LameBoy Display + Copper Wire
