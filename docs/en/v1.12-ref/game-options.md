## Cheat Codes

In tactical screen, press ctrl and keep it not releasing, enter: `GABBI`, then cheat mode activate now.

On tactical screen

- `Alt + B` - spawns an enemy character under the mouse cursor
- `Alt + C` - spawns a civilian character under the mouse cursor
- `Alt + I` - spawns an item under the mouse cursor (usually a firearm)
- `Alt + K` - spawns a cloud of combat gas under the mouse cursor
- `Ctrl + O` - spawns a a Crepitus under the mouse cursor

- `Alt + D` - regenerates Action Points to a selected character
- `Alt + R` - reloads the currently used weapon (works on firearms only)
- `Alt + E` - reveals all the enemies and non-playable characters in the area
- `Alt + T` - teleports selected merc into the cursor's location

- `Alt + 2` - character transforms into a larval Crepitus
- `Alt + 4` - character sits in a wheelchair
- `Alt + 5` - character transforms into an adult Crepitus

- `Ctrl + U` - heals the entire squad

- `Alt + O` - kills all the enemies enemies in sector
- `Ctrl + H` - harming a character below the mouse cursor

- `Alt + W` - changes the item in primary hand, cycling through all items in the game (tactical screen)

- `Ctrl + T` - three mercenaries from the squad are taken prisoners by enemy forces

- `Alt + Enter` - abort enemies turn

On the map screen

- `Ctrl + T` - teleports a squad to a sector under the mouse cursor
  - select a squad
  - plot trave route, move mouse to target sector
  - then press `Ctrl + T`

- `Alt + Auto Resolve` - Kill all enemies in sector

On the laptop screen

- `+` (plus on keyboard) - adds 100 000 $ to funds
- `-` (minus on keyboard) - takes away 10 000 $ from the funds

Space plus click on a mercenary - makes a mercenary join the squad.
